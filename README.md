**🎓 Facial Recognition-Based Smart Attendance System with Analytics and Notification**



📌 Overview



The \*\*Facial Recognition-Based Smart Attendance System\*\* is an AI-powered solution designed to automate the attendance process using advanced computer vision techniques.

It eliminates manual effort and prevents proxy attendance by accurately identifying individuals through facial recognition.



The system captures real-time images via a camera, detects faces, recognizes individuals, and automatically records attendance in a structured database.



\---



🚀 Key Features



\* 🎯 Real-time Face Detection\*\* using RetinaFace

\* 🧠 Accurate Face Recognition\*\* using ArcFace

\* 📸 Automated Attendance Marking\*\*

\* 🗂️ Database Storage\*\* of attendance records

\* 📊 Dashboard for Visualization \& Analytics\*\*

\* 📧 Email Notifications\*\* using SMTP integration

\* 📥 Downloadable Attendance Reports\*\*

\* 🔐 Prevents proxy or fake attendance

5\. Mark attendance automatically

6\. Store data in database

7\. Display records in dashboard

8\. Send notification via email
